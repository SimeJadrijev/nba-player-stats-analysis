# 🏀 NBA Player Stats Analysis

Exploratory data analysis of **NBA player statistics (1996–2022)** using Python.  
The project explores player performance trends, career longevity, physical attributes, and draft outcomes through statistical analysis and visualization.

## 📊 Dataset

**Source:**  
https://www.kaggle.com/datasets/justinas/nba-players-data

The dataset contains historical NBA player statistics including:

- Player bio data
- Physical attributes
- Draft information
- Career statistics
- Team affiliations
- Performance metrics

## 🔎 Topics Covered

### 🧹 Data Cleaning & Preprocessing
- Handling missing values
- Data type conversions
- Feature engineering
- Preparing data for statistical analysis

### 📈 Statistical Distributions
- Histogram analysis of key metrics
- Log transformations
- Overview of statistical distributions across player statistics

### ⏳ Players' Longevity
- Career length analysis
- Distribution of NBA career duration
- Predicting player longevity

### 📏 Physical Attributes & Performance
- Height categorization
- Performance comparison across height groups
- Analysis of shortest and tallest players in NBA history

### 🎂 Age & Performance
- How age impacts performance metrics
- Career trajectory patterns
- Peak performance analysis

### 🎯 Draft Position Analysis
- Relationship between draft position and performance
- Points Per Game (PPG) trends
- True Shooting Percentage (TS%)

### 🌍 College & Nationality
- Impact of college background on draft position
- National vs international player comparisons
- **Mann–Whitney U statistical test** for draft differences

### 🆚 Player & Team Comparisons
- Head-to-head player comparisons
- Career trajectory visualization
- Team-based performance analysis

### ⭐ Dominant Players & Playmakers
- Identifying top performers across eras
- Scoring leaders
- Playmaking analysis
